Game
  Methods
    init()          // get the canvas and canvas context, also start the loading of the images / audios needed
    draw()
    update()
    loadState()     // make all the configuration needed to show the new "state" (??)

  Properties
    animationFrame  // used to start and stop a game
    state           // could be one of the following ['loading', 'menu', 'level'] ++


Level / Map
  Methods
    init()
    draw()
    update()
    start()         // start the level after all needed stuff is loaded
    stop()          // stop the level (need to handle a pause and unpause)
    nextWave()

  Properties
    enemies         // a multidimensional array with X waves and X enemies per wave with their types
    initialGold
    initialLives
    sprite          // background and maybe other images needed
    currentWave
    state           // could be one of the following
                    [
                      'waiting' // waiting for the user to call the next wave
                      'wave'    // 
                    ]


Enemy
  Methods
    init(type)      // the type will be the identifier, so the other properties will be the same if the type is the same
    draw()
    update()
    moveTo(x, y)    // based on the aStarPathFinding
    die()

  Properties
    positions       // x, y
    dimensions      // width, height
    sprite
    type
    hp
    name
    speed
    power           // number of lives to get
    value           // gold earned when he is killed
